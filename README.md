# DataX - Project for Hack The North 2021

[View Video Demonstration](https://www.youtube.com/watch?v=6frm8sA51oc)

## Getting started
- You can run either the frontend, hardware or the Backend code

### Run Frontend
- run yarn on your terminal
- run yarn start
- run yarn android

### Run Hardware
- cd into hardware

### Run Backend
- cd into backend
